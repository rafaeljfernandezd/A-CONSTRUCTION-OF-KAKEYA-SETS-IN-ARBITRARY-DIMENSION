These are the minimum packages required for the notebook to run:
numpy==2.3.1,
plotly==6.7.0,
matplotlib==3.10.3,
scipy==1.16.0

The files run without need to install any packages if they are run in Google Colab, where it was originally created.

All code was created with the aid of Gemini based on the proofs detailed in the associated paper.

The repository contains two Jupyter notebooks implementing interactive visualizations of the same abstract higher-dimensional Perron tree construction. They differ solely in the choice of polytopes employed in the fair subdivision of the pyramid's base: one for the cube, and 
the other for the 2-simplex.
